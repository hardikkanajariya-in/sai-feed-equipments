# Sai Feed Equipment image assets

Images in this folder are placed using the naming expected by the Astro data files:

- `products/` for product detail pages and product cards.
- `solutions/` for solution pages and home sections.
- `gallery/` for the gallery page.
- `brand/` for logo or favicon assets if the final client logo is provided.

The current code uses the supplied product and brochure-derived images already available in this package. Replace files with final client-approved images using the same filenames to keep routes and components unchanged.
